# E-Commerce Sales Analysis

This project analyzes an e-commerce dataset using Python, Pandas, and Power BI.

## Project Overview
The goal of this project is to analyze sales data and identify insights related to revenue, profit, customer behavior, and regional performance.

## Tools Used
- Python
- Pandas
- Power BI

## Dataset
The dataset contains **20,000+ e-commerce sales records** including:

- Order ID
- Customer ID
- Sales
- Profit
- Discount
- Category
- Region
- Order Date
- Ship Date

## Key Analysis Performed
- Data cleaning and preprocessing
- Feature engineering
- Exploratory Data Analysis (EDA)
- Sales and profit trend analysis
- Customer segmentation analysis

## Key Insights
- Technology category generates the highest sales
- West region produces the highest profit
- High discounts reduce profit margins
- A small number of customers contribute to a large share of revenue

## Power BI Dashboard
The Power BI dashboard provides interactive visualizations including:

- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales by Customer Segment
- Profit vs Discount
